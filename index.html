Interactive Beatbox

The interactive beatbox project, as shown in the image, is a web-based application that allows users to create beats by pressing keys on their keyboard. Each key (A, S, D, F, G, H, J, K, L) corresponds to a different sound or beat. The application visually highlights the keys when pressed, making it easy to see which beats are being played in real-time. This project combines web development and audio processing to provide an engaging and intuitive beat-making experience.

Weather Web App

The weather web application is a fully functional tool that provides live weather updates for any city in the world. Users can enter the name of a city into the search bar, and upon clicking the "Search" button, the application retrieves and displays the current temperature, minimum temperature, and maximum temperature for the selected location. Additionally, the app shows a weather icon representing the current weather conditions (e.g., sunny, rainy, cloudy).
